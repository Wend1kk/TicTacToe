# ♟️ TicTacToe AI

[![C#](https://img.shields.io/badge/language-C%23-blue.svg)](https://docs.microsoft.com/en-us/dotnet/csharp/)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

Класична гра "Хрестики-нулики" з інтелектуальним противником на основі алгоритму **Minimax** з оптимізацією **Alpha-Beta Pruning**.

## 🎮 Особливості

- 🔹 **Три рівні складності**:
  - 🟢 **Easy** — випадкові ходи з елементами блокування
  - 🟡 **Medium** — гібридний підхід (правила + обмежений мінімакс)
  - 🔴 **Hard** — ідеальна гра, виграти неможливо (тільки нічия)

- 🔹 **Оптимізований алгоритм** Minimax з Alpha-Beta відсіканням
- 🔹 **Консольний інтерфейс** з інтуїтивною нумерацією клітинок
- 🔹 **Чиста архітектура**: логіка ШІ відокремлена від інтерфейсу
- 🔹 **Легко розширювати**: можна додати нові стратегії без зміни ядра
